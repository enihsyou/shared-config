这是一组可共享的 ESLint 配置，用于控制项目中的编码质量。

## 安装

```bash
npm install --save-dev @enihsyou/eslint-config
```

或者运行以下命令引用 GitHub 上的最新版本。

```shell
pnpm add --save-dev github:enihsyou/shared-config#path:/packages/eslint-config
```
